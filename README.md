# Airdrop Project

This is a ReactJS-based airdrop management project. The project helps track and search for potential airdrop opportunities in the cryptocurrency space.

## Goals

- Provide a simple user interface to track airdrop programs.
- Allow users to search for potential airdrop projects.
- Display detailed information about current airdrop programs, including requirements and rewards.
- Integrate AI to search and classify potential airdrop opportunities.

## Setup

To set up and run the project, follow these steps:

### Prerequisites

- Node.js (version 14.x or newer)
- npm or yarn (yarn is recommended)

### Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/airdrop-project.git
    cd airdrop-project
    ```

2. **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3. **Run the application**
    ```bash
    npm start
    # or
    yarn start
    ```

4. Open the app in your browser at `http://localhost:3000`.
